متجر الكتروني  | E-commerce website



متجر الكتروني متكامل مبني على تقنية بايثون دجانجو وهي من افضل التقنيات لبناء المواقع .
المنتجات مقسمة الى اقسام والى شركات او ماركات
كل منتج له :
    - وصف للمنتج
    - تقييمات ومراجعات
    - tags ( كلمات مفتاحية )
    - يمكن إضافة فيديو للمنتج
    وغيرهم ....

الميزات الرئيسية للمتجر :
    - به نظام كوبونات الخصم
    - متاح وسائل دفع متعددة مثل :  Visa , Paypal , Stripe , الخ..
    - تعدد اللغات مع اختلاف اتجاه الموقع عند تغيير اللغة ( left-to-right / right-to-left )
    - متجاوب مع جميع الاجهزة والشاشات ( Responsive )
    - تسجيل حسابات مستخدمين
    -  تسجيل حسابات تجار
    -  امكانية البحث في الموقع
    -  اضافة المنتجات الى السلة فورا بدون اعادة تحميل الصفحة ( Realtime ) وغيرها من الاشياء التي تتم بنفس الفكرة
     - اضافة المنتجات الى قائمة المفضلة ( Wish list )
    -  تم إنشاء API للموقع ليعمل من خلاله تطبيق للجوال
    - به لوحة تحكم سهلة التعامل
    - متاح إضافة لوحة تحكم بها احصائيات الموقع

وغيرها من الميزات المتنوعة التي لا استطيع حصرها في هذه الصفحة...

كما انني احدث الموقع باستمرار واقوم باضافة له ميزات اخرى

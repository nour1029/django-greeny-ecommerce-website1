SASS Folder connect -

First you need to open your html file in vscode editor. Then you have to install the "live sass compiler" extension from the vscode extension. After installing you can change your compilar path like this.

Then a button will appear at the bottom of the vscode editor with "watch sass" written. Clicking on that button will automatically connect the sass file to your html project.

Quick Changes color, font, & others -

If you want to change your color, font, etc from sass file then you can easily change everything in variable.scss file.

And if you want to change from css file then you can easily change from main.css file.

Use the elements of elsewhere -

Each element, component, card of this template has been created separately so that you can use it elsewhere. Go inside the sass file and you will find all the components in the component, element, layout folders arranged separately.
